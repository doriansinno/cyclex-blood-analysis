# Cyclex

Cyclex is a web application that allows users to input blood values and compare them to reference ranges, with visual feedback and learning features.

## Live Demo
https://cyclex.onrender.com

Note: Access is protected with a password due to the use of external content for demonstration purposes.  
Password: cyclex123

## Features
- Input and analyze blood values
- Comparison with reference ranges
- Data visualization using charts and value scales
- PDF export of results
- Learning section with flashcards and improvement tips

## Tech Stack
- JavaScript / HTML / CSS
- Charting library (for data visualization)
- PDF generation
- Basic application logic for data processing

## How it Works
Users enter their blood values along with additional information such as gender .  
The system compares the values to predefined ranges and displays results visually through charts and structured UI components.

## Purpose
This project was built to explore how medical data can be processed and presented in a clear and user-friendly way.

## Limitations
This project is for educational purposes only and does not provide medical advice.  
Some content is based on publicly available sources and used for demonstration purposes.

## Language
The application interface is currently in German, as it was developed as part of a school project.  
The functionality and logic remain fully applicable regardless of language.
